What is HTML?

How HTML Works

HTML uses a system of tags and attributes to structure and present content on the web. Tags are enclosed in angle brackets. Most HTML elements have an opening tag and a closing tag, though some (like images and line breaks) are self-closing.

Web browsers read these tags and use them to render content visually. For example, the <h1> tag displays large, bold headings, while the <a> tag creates a hyperlink. Attributes provide extra information about an element, like the source of an image or the destination of a link.

Common HTML Tags

  • <html>: Defines the HTML document.
  • <head>: Contains metadata such as the title and character set.
  • <title>: Sets the title of the web page (appears in the browser tab).
  • <body>: Contains the content of the webpage.
  • <h1> to <h6>: Headings, with <h1> being the highest.
  • <p>: Defines a paragraph of text.
  • <a>: Creates hyperlinks to other pages.

HTML Attributes

HTML elements can have attributes that provide additional information. For example:

  • href: Defines the URL of a link.
  • src: Defines the source of an image.
  • alt: Provides alternative text for images.
  • class: Applies CSS styles to elements with a specific class.
  • id: Uniquely identifies an element within the document.
